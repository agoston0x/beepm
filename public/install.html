install your node

your beepm node is where OCR runs and your readings live. nobody else can read them — not us, not the gateway. you deploy it, you own it.

beta · VPS only

during beta, your node must run with a public HTTPS URL — a VPS, EC2, fly.io, or similar. the telegram mini app cannot reach servers inside your home network.

v2 will support fully local (home) install via a gateway relay. coming soon.

1 · deploy the node

docker recommended

# on your VPS / EC2 / fly.io docker run -d \ --name beepm-node \ -p 3064:3064 \ -v ~/.beepm:/data \ -e GATEWAY_URL=https://beepm-gateway.claws.page \ agoston0x/beepm-node:latest # check curl http://localhost:3064/health

from source

# clone on your VPS git clone https://github.com/agoston0x/beepm-node cd beepm-node npm install # python OCR deps pip install opencv-python-headless numpy node bin/start.js

2 · expose HTTPS required

point a subdomain at your VPS (or use caddy/nginx) so the mini app can reach it over HTTPS:

# Caddy (auto HTTPS) your-beepm.example.com { reverse_proxy localhost:3064 }

or use a tunnel while testing:

# cloudflared cloudflared tunnel --url http://localhost:3064 # ngrok ngrok http 3064

3 · pair with the mini app

  • visit https://your-beepm.example.com — you'll see a pairing QR
  • open your phone camera → scan the QR
  • telegram opens the beepm mini app with your node pre-filled
  • sign up with a passkey → dynamic creates your wallet
  • mint your access INFT (gas sponsored, you pay nothing)
  • done. capture your first reading.

daily use

  • open mini app in telegram → tap camera
  • photo goes straight to your node (encrypted in transit)
  • your node OCRs it, stores locally, calls 0G agent via our gateway
  • agent reply appears in the mini app
  • full dashboard at beepm.claws.page/dashboard (sign in with wallet)

troubleshooting

mini app says "can't reach node"

your HTTPS URL isn't resolving or the cert is self-signed. telegram requires a real HTTPS cert. use caddy or cloudflared.

OCR returns garbage

good lighting, camera flat to screen, fill most of the frame. seven-segment displays only for v1.

INFT mint failing

gas is sponsored — you don't need OG tokens. if it fails, check gateway health: curl https://beepm-gateway.claws.page/health
