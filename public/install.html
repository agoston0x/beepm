install zeroclawd

zeroclawd is the daemon that runs on your machine. it stores your encrypted readings and runs OCR locally. pick your install method.

prerequisites

• a machine running linux/macOS (VPS or local)
• docker OR node 18+
• a telegram account

pick a method

docker
npm
source
# pull & run docker run -d \ --name zeroclawd \ -p 3064:3064 \ -v ~/.zeroclawd:/data \ agoston0x/zeroclawd:latest # check curl http://localhost:3064/health

open http://localhost:3064 in your browser → you'll see the pairing QR.

# install npm install -g zeroclawd # install ssocr (for OCR) sudo apt install ssocr # debian/ubuntu brew install ssocr # macOS # configure zeroclawd init # prompts: port, data dir, gateway url # run zeroclawd start
# clone git clone https://github.com/agoston0x/zeroclawd cd zeroclawd npm install # python deps for OCR pip install opencv-python-headless numpy # run node bin/zeroclawd.js init node bin/zeroclawd.js start

expose to telegram

zeroclawd needs to be reachable from your telegram mini app. for local dev use ngrok, cloudflared, or tailscale funnel:

# ngrok ngrok http 3064 # cloudflared cloudflared tunnel --url http://localhost:3064

copy the public URL — you'll enter it in the pairing step.

pair with telegram

  • open the beepm mini app in telegram → t.me/beepm_telegram_bot/beepm_tg_app
  • sign in with email or passkey (dynamic creates your wallet)
  • mint your access NFT (one-time)
  • tap "pair zeroclawd" and scan your instance QR (or paste the URL)
  • done. capture your first reading.

troubleshooting

OCR not finding digits

ensure ssocr is installed and in PATH. check zeroclawd logs.

can't reach instance from telegram

zeroclawd must be publicly reachable. use ngrok/cloudflared or deploy to a VPS.
